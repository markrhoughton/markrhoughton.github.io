3D Numerical Model Development

  • University of Leeds

Developed a 3D numerical model in C using tools from the PETSc library to assemble and solve large sparse matrices, leading to novel mechanical property predictions for large random networks. Implementation was split into two iterations, the first of which combined C driven matrix assembly with MATLAB solvers. The second model iteration was written purely in C with calls to the PETSc library. Simple bash scripts were used throughout the project to automate aspects of workflow and streamline property analysis.

Lattice Based Network Generator Development

  • University of Leeds

Implemented a 2D node driven lattice based fibre network generator, for aperiodic 2D triangular meshes. Added functionality for mesh dilution, allowing me to study rigidity transitions and validate my numerical model against literature. Added periodicity support for diluted meshes, significantly reducing finite size effects.

2D Numerical Model Prototype Development

  • University of Leeds

Prototyped a numerical model leading to successful prediction of the mechanical properties of 2D random fibre networks. To do this I divided the implementation into: model read/write of network data; local matrix/vector assembly; global system assembly from local information; solving a global system of equations; updating network data; and performing post-solve analysis.
